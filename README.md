# Quartz Clickable Images

A Quartz v5 plugin that makes images clickable and opens them in a beautiful lightbox/zoom modal.

This is an updated and improved fork of [vazome/quartz-clickable-images-zoom-plugin](https://github.com/vazome/quartz-clickable-images-zoom-plugin) made compatible with Quartz 5.

## Features

- Click any image to open in a full-screen lightbox
- Smooth zoom animation
- Keyboard support (ESC to close)
- Mobile-friendly
- Lightweight and easy to use

## Installation

```bash
npx quartz plugin add github:fardm/quartz-clickable-images
```
